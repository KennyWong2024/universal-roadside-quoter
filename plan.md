Para no saturarnos, programaremos en este orden:

Fase 0: Cimientos

Configurar Firebase Project y variables de entorno.

Crear estructura de carpetas vacía.

Fase 1: Identidad (El Portero)

Implementar AuthService.

Crear colección allowed_users manual en Firebase.

Probar flujo: Login -> Check Whitelist -> Home.

Fase 2: El UI Kit (El Cascarón)

Configurar Tailwind.

Crear Layout base (Sidebar + Header + Content).

Crear componentes básicos (Botón, Card).

Fase 3: El Motor (Calculadora)

Programar la lógica de precios en TypeScript puro (sin UI).

Crear la interfaz de usuario del cotizador.

Fase 4: Persistencia

Guardar cotizaciones en Firestore.
